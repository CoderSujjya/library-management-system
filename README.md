# Library Management System

A simple Library Management System built using Spring Boot, Spring MVC, JPA/Hibernate, MySQL, and Thymeleaf.

## Features
- Add, update, delete, and view books
- Manage library members
- Track issued and returned books
- Simple and clean web interface using Thymeleaf

## Tech Stack
- Java
- Spring Boot
- Spring MVC
- Spring Data JPA / Hibernate
- MySQL
- Thymeleaf
- Maven

## How to Run
1. Clone the repository

2. Create a MySQL database
```sql
   CREATE DATABASE library_management_system;
```
3. Update `application.properties` with your own MySQL username and password
4. Run the project using your IDE (Eclipse/IntelliJ) or with:
5. Open your browser and go to:
6. http://localhost:2121
